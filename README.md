# 上海海港2026赛季赛程日历 📅

## 简介

这是上海海港足球俱乐部2026赛季中超联赛的赛程日历文件，可以导入到手机日历应用中。

## 使用方法

### 方法1: 直接订阅（推荐）
复制以下链接到你的日历应用中：
```
https://[your-username].github.io/shanghai-port-calendar/shanghai_port_2026.ics
```

### 方法2: 下载导入
1. 点击 `shanghai_port_2026.ics` 文件
2. 下载到手机
3. 用日历应用打开导入
