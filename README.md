## 📱 如何使用日历

### iOS用户
1. 复制订阅链接: `https://glenman.github.io/shanghai-port-calendar/2026-calendar.ics`
2. 打开"设置" → "日历" → "账户" → "添加账户" → "其他"
3. 选择"添加已订阅的日历"，粘贴链接

### Android用户
1. 下载 [2026-calendar.ics](2026-calendar.ics)
2. 用日历应用打开，选择导入

### Google Calendar
1. 打开Google Calendar
2. 左侧"其他日历" → "+" → "从网址添加"
3. 粘贴订阅链接
